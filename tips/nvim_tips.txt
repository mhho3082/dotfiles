Space bar: leader key
(most keymaps can be explored with the which-key popup)

K - LSP details
J - LSP diagnosis
gr - LSP rename

ga... - align stuff
(See junegunn/vim-easy-align)

gc... - line comment
gb... - block comment

C-w + [+-<>] - resize windows
C-w + [v/s]  - split

C-v - visual block

"+y - yank to clipboard
"+p - paste from clipboard

:new term://fish - New terminal split
<C-\><C-n>       - Get out of terminal mode

<C-n> / <C-p> - next / prev (in autocomplete)

<A-[hjkl]> - move selection

For nvim-tree: See kyazdani42/nvim-tree.lua
