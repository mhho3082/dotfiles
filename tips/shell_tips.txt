Basic
uname -a - basic system info
inxi     - detailed system info
htop     - cpu and ram loads

The `ls*` commands
lscpu    - cpu info
lsmem    - mem info
lsusb    - usb info
lspci    - pci devices info

Misc.
sensors  - sensor data (e.g., temperature)
acpi     - battery level
date     - current time

Some commands:

Find largest directories in ~/
du ~ -h -d 1 | sort -hr
